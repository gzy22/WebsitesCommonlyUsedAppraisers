## 使用方式

直接打开 `index.html` 即可使用，无需任何服务端环境。

- **搜索**：输入网站名称或功能描述，实时筛选
- **分类导航**：点击导航栏标签，快速跳转到对应分类
- **网站卡片**：点击卡片在新标签页中打开目标网站
